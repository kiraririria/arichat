üì¶ Installing Arichat Classic

Arichat Classic is a Minecraft mod that adds AI functionality directly into the game.

Requirements:

  • Minecraft 1.12.2
  • Forge
  • Java 8+
  • Mappet and other dependencies
  • Internet access or local model

Installation Steps:

  1. Download the latest version of Arichat Classic from the releases page
  2. Place the .jar file in the mods folder of your Minecraft client
  3. Add Mappet and other required McHorse mods
  4. Launch Minecraft
  5. Configure the mod through the menu
